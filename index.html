TrndStar
Commissioned Project
"TrndStar aims to bring new features to the social media environment. Currently, there is not a good environment for new upcoming artists to make a name for themselves. TrndStar plans to put these upcomers right in the middle of the action.
Users can gain a following by uploading videos, competing, and collaborating. Users can challenge other users and have their videos seen by both users followers. Multiple users can collaborate and create one video to attract as many followers as possible.
TrndStar features the ability to upload 90 second videos and gain exposure as a new artist or entertainer."
- Official TrndStar statement
iOS Food Tracker
Hobby Project
This app was my introduction to iOS development and was centered around this tutorial from apple. This app shows a list of meals, including a meal name, rating, and photo. A user can add, remove, or edit a meal. To add a new meal or edit an existing one, users navigate to a different screen where they can specify a name, rating, and photo for a particular meal.
r/dailyprogrammer challenges
Hobby Project
Dailyprogrammer is about challenging programmers of all skill level with weekly programming challenges. There are 3 challenges a week that are posted at increasing difficulty. Solutions are peer reviewed and redditors can ask the community for feedback and comments. I set out to complete these challenges using many different languages in order to expand my knowledge. You can find more information about this on their reddit wiki page.
